## **Actividad en Clase: Explorando Motores de Bases de Datos**  

⏳ **Duración total: 60 minutos**  
🎯 **Objetivo:** Identificar y comparar distintos motores de bases de datos según sus características, ventajas y usos.  

---

1. **Grupos de 4 máximo**
2. A cada grupo se le asigna **un motor de base de datos** (Ejemplos: MySQL, PostgreSQL, MongoDB, Redis, Neo4j).  

📋 **Preguntas clave para investigar:**  
✅ ¿Es una base de datos relacional o no relacional?  
✅ ¿Cómo almacena y organiza los datos?  
✅ ¿Cuáles son sus principales ventajas?  
✅ ¿Para qué tipo de aplicaciones es más útil?
